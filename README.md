## Usage
### Compile the demo
```bash
make clean
make
```
### Compile the TCP part
```bash
make clean
make TCP
```
### Compile the UDP part
```bash
make clean
make UDP
``` 
### Compile the Multi-thread TCP part
```bash
make clean
make Mul
```
### Compile the Web Server
```bash
make clean
make Web
```
### Run the bin
- ___TOTALLY THE SAME AS WHAT IN___ [this pdf](./lab1-cs536.pdf)
